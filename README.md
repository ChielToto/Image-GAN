# GAN for Images
Using Generative Neural Networks (GANs) to create images from images. 

Current version does not produce functioning results as finding the right balance between the two networks deemed harder than anticipated. Nowadays, better results can be achieved through Diffusion Models. 

I got it working eventually on some drawings I had made, using the image_multiplier to increase the size of my dataset through alterations in the images. The GAN overfit onto a single drawing but produced it nonetheless.
